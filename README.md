# Ashish Bhargava - Online Profile

This is a complete, static web application designed to showcase Ashish Bhargava's IT training and data analysis portfolio. It was designed with a striking, premium "technological theme" featuring glassmorphism elements, neon glows, and interactive particles.

## Quick Start
To view this profile:
1. Double-click the `index.html` file to open it in any modern web browser.
2. Alternatively, you can use a local web server (e.g. `npx serve .` or `python -m http.server 3000`) for the best experience.

## Hosting Online
Because this profile consists entirely of standard web files (HTML, CSS, JS), it is exceptionally portable. You can host it online for free using any static hosting service in minutes:
- **GitHub Pages:** Upload the `online-profile` directory to a new repo, go to Settings -> Pages, and select your main branch.
- **Vercel / Netlify:** Drag and drop the `online-profile` directory onto their web interfaces to deploy a live URL instantly.

## Features Included
* **Fully Responsive:** Adapts flawlessly to mobile devices and wide screens.
* **Modern Aesthetic:** Deep tech theme with gradient glowing rings, custom typography (`Orbitron` & `Inter`), and interactive particle backings.
* **Scrollspy Navigation:** Intelligent, sticky top navigation that highlights sections as the user scrolls.

## Files
* `index.html`: The markup structure containing extracted resume text.
* `styles.css`: The styling engine making use of CSS Variables and `backdrop-filter`.
* `script.js`: Interactive elements configuration (particles).
* `assets/`: Contains AI-generated technological background and professional cyber avatar.
